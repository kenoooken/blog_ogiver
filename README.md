# アプリの用途
『ブロガーに寄り添った』アプリです。

・ドメインパワーが弱い。
・閲覧してもらえない。
・収益化ができない。

など悩みを抱えているブロガーが多く存在します。
そんなブロガーの悩みを解決すれば、サービスを利用してくれてコンテンツが増え、自動的にサービスが回り出す。と考えています。

対象ユーザーを、
・コンテンツ制作者のブロガー
・検索流入からの閲覧者

としています。
